# 株価ダッシュボード (S&P 500 & All Country)

このアプリケーションは、S&P 500と全世界株式（All Country）の現在の株価と過去の推移を簡単に確認できるダッシュボードです。

## 主な機能

*   **自動更新**: 画面を開いておくだけで、5分ごとに自動的に最新の株価情報に更新されます。
*   **2つの主要指標**: S&P 500とAll Country（ETFであるACWIを代用）の現在の価格を表示します。
*   **株価の推移グラフ**: 過去の株価推移をグラフで視覚的に確認できます。
*   **表示期間の切り替え**: グラフの表示を「日次（毎日）」「月次（毎月）」「年次（毎年）」から選んで切り替えることができます。

---

## 実行方法（初めての方へ）

このアプリを動かすためには、Pythonというプログラミング言語の環境が必要です。以下の手順に沿って準備と実行を行ってください。

### 1. 準備（初回のみ）

まず、アプリを動かすための部品（ライブラリ）をインストールします。
ターミナル（Windowsの場合はコマンドプロンプトやPowerShell）を開き、このフォルダに移動して、以下のコマンドを入力してEnterキーを押してください。

```bash
# 必要な部品をインストールします
pip install -r requirements.txt
```

### 2. アプリの起動

インストールが終わったら、以下のコマンドを入力してEnterキーを押すと、アプリが起動します。

```bash
# アプリを起動します
streamlit run app.py
```

### 3. アプリを見る

コマンドを実行すると、自動的にブラウザ（ChromeやSafariなど）が開き、ダッシュボード画面が表示されます。
もし自動で開かない場合は、ターミナルに表示されている `Local URL: http://localhost:8501` という文字をコピーして、ブラウザのアドレスバーに貼り付けてアクセスしてください。

### アプリの終了方法

アプリを終了したい場合は、ターミナルで `Ctrl + C`（コントロールキーを押しながらCキー）を押してください。

## データ元について
このアプリの株価データは、「Yahooファイナンス」の無料公開データを利用しています。実際の取引価格とはわずかな遅れが生じる場合がありますので、参考情報としてご活用ください。
